# JS 面向对象之旅

> 用 ES6 入门面向对象编程，语法甜美，而且直接面向实用

**为谁而写？**

对编程语言有基本的认识，起码知道函数，参数这些概念，但是对面向对象编程一无所知的新手。写过很多年 jQuery ，但是其实写前端一直没怎么用面向对象思想的开发者。

**内容简介**

类和对象的概念图解代码小例子演示。为后续上手摩登 JS 技术和框架，例如 Nodejs 和 React 打下基础。
